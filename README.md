# zillowParser

## Usage

### Build and Install Packages
```bash
$ curl https://raw.githubusercontent.com/ilyakmet/zillowParser/master/install.sh | sh
```

### Run Min Parser
```bash
$ zillow-minparser
```

```bash
$ zillow-maxparser
```
